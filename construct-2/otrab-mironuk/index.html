Шипы

При касании с Шипами (on collision with) Сонику наносится урон.

Предварительно для этого нужно будет создать переменную:

  1. Нажми правой кнопкой на пустое пространство
  2. Add global variable
  3. Пишем название - например "жизни".
  4. Значение по умолчанию (Default value) ставим в 100
