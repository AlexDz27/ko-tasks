Задания

Мигание при уроне

Можно использовать экшен (action) Flash - тогда Игрок будет мигать какое-то время.

Для этого нужно предварительно создать (подключить) поведение Flash у Cоника (Игрока).

Прыгалка

Чтобы сделать "прыгалку", используй картинки из Урока 4. Также не забудь сделать анимацию для прыгалки.

Проигрыш в игре

Сделай проигрыш в игре - появляется текст с надписью "Проиграл", если здоровье меньше или равню нулю.

Победа в игре

Сделай победу в игре - появляется текст с надписью "Победа", если ты дошёл до какого-то спрайта, который в твоей игре означает победу (например, картинка с мужиком из папки "Урок 3").

Прыгать сквозь платформы вверх (находясь снизу)

Чтобы сделать платформы, сквозь которые можно прыгать, находясь снизу, используй поведение Jumpthru.

Самый нижний уровень (платформа) с шипами

Сделай сплошные шипы на самом нижнем уровне в твоей игре.
